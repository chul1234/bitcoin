프로젝트 목표
Gemini API의 '실시간 웹 검색' 기능을 활용해 가장 빠른 글로벌 코인 속보를 수집 및 분석하고, 이를 업비트(Upbit) 실시간 데이터와 융합하여 코인의 미래 방향성을 예측하는 사용자 맞춤형 가상 투자 플랫폼을 구축합니다.
핵심 서비스 파이프라인: [초기 가상 자산 설정] -> [Gemini 실시간 미국 뉴스 검색 및 번역] -> [AI 호재/악재 판별 및 방향성 예측] -> [업비트 실시간 시세 대조] -> [사용자 가상 매매 실행]
서비스 및 주요 기능
사용자 정의 투자 시스템
- 자율적 시드머니 
1. 가상 자산
2. 사용자가 직접 원하는 초기 자산을 입력

- 실시간 매매 
1. 업비트의 실시간 호가 데이터 바탕
2. 가상 매수/ 매도 진행
3. 수량 입력시 즉시 예상 체결가와 필요 금액 계산
4. 보유 잔고가 연동 되어 실시간 변동
시간대별 시장 통계 분석
1. 거래량/ 변동성 패턴 추적업비트 데이터를 분석 24시간 중 거래량이 가장 활발한 시간, 침체된 시간 산출
2. 시간대별 매매 적기 도출“현재 시간은 과거 통계산 변동성이 15% 이상 증가 구간”과 같이 안내 추가
3. 코인별 활성화 시간대 대조 코인마다 집중적으로 거래가 일어나는 시간대 분석 -> 리스트
Gemini AI 실시간 검색 및 방향성 제시 
1. Gemini API의 google search Grounding 기능 활용 -> 미국 현지 메체의 영문 속보 실시간 확인
2. 분석 파이프 라인 : 수집된 기사 별도의 번역 api없이 gemini 내부에서 1번 호출 **[영문 기사 검색 -> 핵심 3줄 한글 요약 -> 시장 호재/악재 방향성 점수(0~100점) 산출]**까지 통합 처리
3. 예측 대조 검증 시스템 :- AI가 예측한 방향성 시그널을 화면에 표시- 10분 ~ 1시간 뒤 업비트 시세 변동분과 1:1로 직접 대조 하여 성공 여부 트래킹
4개의 테마, AI 포트폴리어 큐레이션 
1. 저가 매수 & 고잠재력 (Low-Cap / High-Potential)현재 단가와 시가총액이 낮지만, 최근 파트너십 등 웹3.0 관련 긍정적 기사가 다수 보도된 알트코인.
2. 고위험 & 고수익 (High-Risk / High-Return)최근 24시간 내 변동성과 거래량이 극심하여 단기 펌핑 수익을 낼 수 있는 코인 (소액 단기 투자 권장).
3. 우량 자산 & 저변동성 (Large-Cap / Low-Volatility)거시 경제 불안정 뉴스 속에서도 가격 방어력이 뛰어난 비트코인, 이더리움 중심의 안전 자산.
4. 인물/내러티브 연결성 (Influencer & Narrative)일론 머스크 등 유명 인사의 발언이나 글로벌 밈(Meme), 정치적 이슈와 강하게 연결되어 실시간 검색량이 폭증하는 테마 코인.
거래소 차트
1. 1분, 15분, 분봉~일봉, 월봉까지 업비트와 완벽히 동일한 캔들스틱 차트 제공
2. 과거-미래 데이터 색상 분리과거~현재 (실제 데이터) -> 상승 : 빨강, 하락 : 파랑미래(ai 예측 데이터) -> 시간 축 대비 되는 색 (상승 : 주황, 하락 보라)
3. 실시간 거래량 막대 그래프 배치 
UX/UI : 최소 클릭, 최소 스크롤 
- 원 스크린 (One-Screen) 레이아웃: 스크롤 바가 생기지 않도록 모니터 크기에 맞춰 좌측엔 차트, 우측엔 매매 폼을 고정 배치합니다.
- 자동 입력 (Auto-fill) 연동: 사용자가 우측 호가창의 특정 가격을 마우스로 클릭하면, 매매 폼의 '주문 가격' 입력란에 해당 숫자가 즉시 자동 기입됩니다.
- 원클릭 퀵 비율 버튼: 직접 수량 타이핑을 최소화하기 위해 매매 폼 하단에 [10%], [25%], [50%], [MAX] 버튼을 배치하여, 클릭 한 번으로 가상 잔고 대비 매수 가능 수량이 세팅됩니다.
- 원클릭 포트폴리오 매수: AI가 추천한 테마별 포트폴리오 팝업 창에서 [이 구성으로 즉시 매수] 버튼을 누르면 비율에 맞춰 분산 투자가 자동 실행됩니다.

Java, Spring Boot 3.x
데이터 파이프라인 및 비즈니스 제어
• WebClient를 통한 업비트 및 Gemini API 비동기 논블로킹 통신.
• @Scheduled 어노테이션을 활용해 서버 백그라운드에서 10분 주기로 Gemini 최신 뉴스 검색 실행.
• 가상 매수/매도 트랜잭션 처리 및 잔고 정합성 검증 서비스 로직.
Frontend
AngularJS, HTML5
SPA 기반 동적 웹 폼 및 렌더링
• 양방향 데이터 바인딩을 활용하여 사용자가 폼에 수량을 입력하는 즉시 화면 상의 예상 체결가/잔고가 변동됨.
• 보유 잔여 현금을 초과한 입력 시 즉각 에러 텍스트를 노출하는 실시간 유효성 검사(Real-time Validation) 구현.
• '입력 대기 $\rightarrow$ 체결 중 $\rightarrow$ 완료' 상태를 시각적으로 보여주는 다중 상태 제출 버튼 적용.

기술 타당성 및 외부 API 연동 정책 검증
upbit 
- REST API (과거 캔들 조회): 초당 최대 10회 허용 (초기 차트 로딩용으로 스펙 충족).
- WebSocket API (실시간 시세 수신): 초당 5회, 분당 최대 100회 데이터 수신 허용.
- 검증 결과: 실제 주문 API가 아닌 Public API를 활용하므로, IP 단위 제한 한도 내에서 1초마다 시세를 갱신하는 퀄리티 높은 대시보드를 무상으로 구축 가능합니다.
gemini API (free tier)
- 스케줄링 최적화: 서버 단위에서 10분 주기로만 뉴스를 검색하도록 요청 빈도를 제어하여, 분당 요청 수(RPM) 및 일일 요청 수(RPD) 무료 한도 내에서 완벽하게 구동 가능합니다.
- 실시간 데이터 접근성: 과거 데이터만 아는 일반 LLM의 한계를 극복하기 위해, 프롬프트 내에 *"기준일: 오늘(현재 시각), 최근 1시간 이내 기사 검색"*을 명시하고 Google Search Grounding 도구를 강제 활성화하여 최신 웹 인덱스 자료만 긁어오도록 구현합니다.
